## Usage
* ### SetUp Requirements
    * Android Studio: version >= 4.0
    * minSDKVersion 21
    * targetSDKVersion 31
    *           kotlinOptions {
                   jvmTarget = "1.8"
                }
* ### SetUp 
      git clone https://github.com/zakariazaim17/travelBuddy.git  via https
      
* **Open project with Android Studio as an existing project, built and run the project with your android device or emulator

## Technical Specifications
* ### Persistence
     * SharedPreference
     * File
     * Network Storage - Firebase
* ** Service**
     * Ktor server For fetching informations using Retrofit
     * Firebase for files uploading and fetching
* **Hardware - Camera**
* ### Architectural Designs
     * ViewModel & LiveData
* ### Libraries Used
     * Material design
     * Coil
     * Lottie
